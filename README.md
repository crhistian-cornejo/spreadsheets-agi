Una suite de productividad donde puedes hablar con tus documentos. Crea y edita Sheets y Docs usando lenguaje natural.

## Lo que puedes hacer

- **Sheets dinámicos**: Pide crear tablas, ordenar o sumar columnas y la IA lo hace por ti.
- **Editor Docs**: Documentos con asistente IA integrado.
- **AI Chat Mode**: Un chat que genera "artifacts" (tablas, documentos) en tiempo real.
- **Diseño**: Minimalista y rápido usando Tailwind v4.

## Desarrollo

```bash
# Instalar todo
bun install

# Modo desarrollo
bun dev
```
